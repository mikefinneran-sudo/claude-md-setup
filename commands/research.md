# Research Command

Create a comprehensive research document on the specified topic.

## Task

Research: {{ARG_1}}

## Instructions

1. **Research Phase**:
   - Use Perplexity MCP (if available) or WebSearch to gather current information
   - Focus on authoritative sources, recent developments, and expert opinions
   - Identify key concepts, trends, and debates

2. **Document Structure**:
   Create a markdown document with the following sections:

   ## Executive Summary
   - 2-3 paragraph overview
   - Key findings and recommendations

   ## Background
   - Context and definitions
   - Why this topic matters

   ## Current State
   - Latest developments
   - Key players and technologies
   - Market landscape (if applicable)

   ## Deep Dive
   - Technical details
   - Use cases and applications
   - Challenges and limitations

   ## Best Practices
   - Implementation guidelines
   - Common pitfalls to avoid
   - Expert recommendations

   ## Future Outlook
   - Emerging trends
   - Predictions and possibilities

   ## Resources
   - Links to key sources
   - Further reading
   - Tools and frameworks

3. **Deliverable**:
   - Save as: `~/Documents/Projects/Research/YYYY-MM-DD - [Topic] Research - v1.md`
   - Professional tone
   - Well-structured with clear headings
   - Actionable insights
   - Properly cited sources

## Style

- Professional and objective
- Technical where appropriate
- Accessible to technical readers
- Focus on practical applications
- Include examples and use cases
